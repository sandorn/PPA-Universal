# Ribbon 手工回归清单（PowerPoint / WPS）

在改动 **Ribbon**（`PPARibbon.xml`）、**回调**（`RibbonCallbacks.cs`）或 **COM 宿主 / 适配层** 后，建议在 **Microsoft PowerPoint** 与 **WPS 演示** 各执行一遍下列检查。勾选表示通过。

更新日期：[当前日期]

## 通用

| 检查项 | PPT | WPS |
| ------ | --- | --- |
| 加载项可加载，**PPA** 选项卡出现在功能区（在「开始」之后） | ☐ | ☐ |
| 选项卡内各组按钮图标正常显示、无占位裂图 | ☐ | ☐ |

## 对齐和分布

| 检查项 | PPT | WPS |
| ------ | --- | --- |
| 多选形状：左对齐、水平居中、右对齐、顶对齐、垂直居中、底对齐 | ☐ | ☐ |
| 对齐参考下拉：选中对象 / 幻灯片 / 第一个对象 / 最后一个对象 与行为一致 | ☐ | ☐ |
| 至少 3 个形状：水平分布、垂直分布 | ☐ | ☐ |

## 吸附

| 检查项 | PPT | WPS |
| ------ | --- | --- |
| 先选基准再选其余：向左 / 右 / 上 / 下吸附，位置符合 supertip 描述 | ☐ | ☐ |
