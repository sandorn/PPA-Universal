## 📝 发布说明模板

```
PPA PowerPoint 插件 v1.0.0
============================

发布日期: 2025年1月

系统要求：
- Windows 7 或更高版本
- Microsoft PowerPoint 2016 或更高版本
- .NET Framework 4.8
- Visual Studio Tools for Office Runtime

主要特性：
- 自定义功能区集成
- 图形对齐和格式化工具
- 批量操作功能（表格、文本、图表）
- 多语言支持（中文/英文）
- 配置化支持（XML 配置文件）
- 全局快捷键支持
- 异步操作支持

安装步骤：
1. 运行 setup.exe
2. 按照安装向导完成安装
3. 重启 PowerPoint
4. 插件将自动加载

配置文件：
插件会在 %AppData%\PPA\ 目录创建 PPAConfig.xml 配置文件，
用于自定义格式化样式和快捷键设置。

卸载步骤：
1. 控制面板 → 程序和功能
2. 找到 "PPA" 并卸载

技术支持：
- GitHub: https://github.com/sandorn/PPA
- Issues: https://github.com/sandorn/PPA/issues
```
